1. 디버깅 -> 내 의도에 맞게 프로그램 실행 확인, 나중에 나타날 문제점은 없는지 생각, 에러를 발생할 확률이 크다.
2. 알고리즘, 자료구조 -> 코딩테스트 필수, 구글검색식으로 갖다붙이고 되네와 차원이 다름, 실행이 깔금
3. 다른 사람의 소스코드 참조 -> github -> 오픈소스
4. 주석 -> 코드 이해 증가,일주일 뒤면 잊어버린다, 다른사람이 봤을때 이해할 수 있도록
